# Student Attendance vs Academic Performance Analysis

## Project Overview
This project explores the relationship between student lecture attendance and academic performance. The goal is to determine whether higher attendance is associated with higher mean marks across different year groups.

The analysis is performed using Python and includes data cleaning, exploratory data analysis, visualisation, and linear regression modelling.

---

## Dataset
The dataset contains the following variables:
- **Year**: Student year group (0–4)
- **Course**: Degree programme
- **Attendance**: Lecture attendance percentage
- **Mean_Mark**: Average student mark

---

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

---

## Key Steps
1. Loaded and inspected the dataset
2. Cleaned attendance data (converted from string to numeric)
3. Performed exploratory data analysis (scatter plots)
4. Visualised data by year group
5. Applied linear regression models for each year group
6. Analysed the correlation between attendance and performance

---

## Key Findings
- No strong correlation was found between attendance and mean marks.
- Regression lines for each year group were mostly flat.
- This suggests attendance alone is not a strong predictor of academic performance in this dataset.
- Patterns were consistent across all year groups.
